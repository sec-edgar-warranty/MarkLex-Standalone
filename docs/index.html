System Requirements

  • At least 4GB RAM (8GB recommended)
  • 500MB free disk space for models
  • Internet connection for initial model download

Getting Started

1

Download & Install

Download the appropriate version for your operating system and install MarkLex Desktop.

2

Setup Models

Launch the app and go to the "Setup" tab to download the required Word2Vec embedding models (~300MB).

3

Create Lexicons

Use the "Lexicon" tab to generate semantically similar terms from seed words. Export results as CSV.

4

Analyze Text

Upload documents in the "Text Analysis" tab to analyze against business dimensions and themes.
